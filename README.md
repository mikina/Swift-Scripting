# Swift-Scripting
Command line applications in Swift. 

To run application inside terminal:

```swift
$ chmod +x hello.swift 
$ ./hello.swift
Hello from terminal!
```

## Find prime numbers

Find prime numbers between 2 and 10000000. Calculate execution time. 

```swift
$ ./prime.swift
Find prime numbers between 2 and 10000000 :: Time: 21.4241750240326
```
